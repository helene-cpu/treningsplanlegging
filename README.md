# Treningsplanlegger

## Beskrivelse

En nettside hvor brukere kan registrere seg, logge inn, lage treningsplaner og loggføre treningsøkter. Løsningen lagrer data i en database og fokus på universell utforming, sikkerhet og dokumentasjon.

## Installasjon

**Oppsett av virtual enviroment:**

python3 -m venv env

source env/bin/activate

pip3 install flask flask_sqlalchemy

**For å kjøre:**

source env/bin/activate

python3 app.py

## Funksjonalitet

- Registrering
- Innlogging
- Lage treningsplan
